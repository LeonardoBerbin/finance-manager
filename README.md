# Finance Manager

Personal finance manager built with modern fullstack architecture.

## Stack

- Frontend: React + TypeScript
- Backend: NestJS + TypeScript
- Database: PostgreSQL
- Auth: JWT

## Requirements

- Docker
- Docker Compose V2 (`docker compose`)
